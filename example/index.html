Flutter Spelling Number


Spelling in English

1. Example Convert Number into words

Code

           
        SpellingNumber(
        lang: "en",
        digitsLengthW2F: 3,
        decimalSeperator: "point"
        ).convert(1625444.13)

Number and Selling value

1625444.13

2. Example Convert Dollars and Cents into words

Code

     

        SpellingNumber(
            lang: "en",
            wholesUnit: "dollars",
            fractionUnit: "cents",
            digitsLengthW2F: 2,
            decimalSeperator: "and"
        ).convert(12300.23);

Number and Selling value

12300.23

3. Example Convert Weight number into words

Code

  
        SpellingNumber(
            lang: "en",
            wholesUnit: "ton",
            fractionUnit: "kg",
            digitsLengthW2F: 3,
            decimalSeperator: "and"
        ).convert(100.951);
